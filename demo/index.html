Count to {{countTo}} from {{countFrom}} over 4 seconds





Example directive code

<h1 count-to="{{model.countTo}}" duration="4" count-from="{{model.countFrom}}"></h1>


With the AngularJS Number Filter

<h1 count-to="{{model.countTo}}" duration="4" count-from="{{model.countFrom}} count-filter="['number', 2]"></h1>

Note: stacking filters is not supported; any stacked filter must be implemented as a single filter for use in this way.
